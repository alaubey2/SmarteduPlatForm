# Smart Education Platform

منصة تعليمية ذكية متكاملة تم تطويرها باستخدام PHP وMySQL وHTML وCSS وJavaScript.

## الميزات:
- محتوى تعليمي ذكي
- فصول افتراضية
- تقييمات ذكية
- مجتمع تفاعلي
- شهادات رقمية
- نظام حضور وغياب
- لوحة تحكم متعددة الأدوار (طالب، مدرس، مشرف)
- نظام إشعارات وتحليلات
